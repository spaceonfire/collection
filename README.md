Framework agnostic PHP collections

## Install

Via Composer

```bash
$ composer require spaceonfire/collection
```

## Usage

```php
$collection = new spaceonfire\Collection\Collection([0, 1, 2, 3]);
//$collection->sum() === 6
```

Read [documentation](./docs/README.md) for more.
